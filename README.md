# GST Knowledge Graph Risk Analytics

AI-powered system for visualizing GST entities, invoices, and risk relationships using a Knowledge Graph.

## 🚀 Features
- Interactive graph visualization
- Risk analytics dashboard
- Audit trail for each entity
- Neo4j graph database
- FastAPI backend
- React frontend

## 🧠 Tech Stack

Frontend
- React
- TailwindCSS
- ForceGraph2D

Backend
- FastAPI
- Neo4j
- Python

## 📊 System Architecture

User → React Dashboard → FastAPI API → Neo4j Knowledge Graph

## 🔍 Capabilities

- Detect high-risk GST entities
- Trace invoice relationships
- Visual audit explanations
- Graph-based risk investigation
